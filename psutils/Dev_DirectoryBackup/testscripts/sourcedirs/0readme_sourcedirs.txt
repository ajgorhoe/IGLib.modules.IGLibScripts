
This directory contains source directories to be used by the test scripts
for directory backups.

BackupSourceBasic/ :
  Basic source directory for testing backups, with a branched structure.

BackupSourceAdditionalFiles/ :
  Source directory for testing backups with similar contents as the ../BackupSourceBasic/ , but with some ADDED files and directories.

BackupSourceMissingFiles/ :
  Source directory for testing backups with similar contents as the ../BackupSourceBasic/ , but with some files and directories MISSING.

BackupSourceMissingAndAdditionalFiles/ :
  Source directory for testing backups with similar contents as the ../BackupSourceBasic/ , but with both some files and directories MISSING,
  AND some ADDITIONAL files.
